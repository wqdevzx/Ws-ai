## 🌌 **PROJECT: Ws-ai (The RAHA Core)**
The **Ws-ai** engine is a high-performance Facebook Messenger automation framework built on the **GoatBot** architecture. It integrates advanced Node.js modules with a focus on aesthetic user interfaces.

### 🛠 **TECHNICAL STACK**
* **Engine:** Node.js v18+ 
* **Framework:** GoatBot / Custom Middleware
* **Graphics:** Canvas API (3D Styled Information Cards)
* **Process Management:** PM2 (Enterprise-grade stability)
* **Styling:** Aesthetic Typography & Gradient UI

---

## ⚡ **CORE CAPABILITIES**
* **Dynamic Command Suite:** Custom-coded modules including `song.js`, `pair.js`, and `top.js`.
* **Visual Excellence:** Automated generation of high-fidelity 3D user cards with custom hex-glow effects.
* **Robust uptime:** Configured on the **Nyxora Main Server** for 24/7 seamless operation.
* **Advanced Logic:** Sophisticated debugging and modular script integration.
